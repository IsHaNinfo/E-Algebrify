## Perfect Square Trinomials

These trinomials fit the following patterns:

`
a^2 + 2ab + b^2 = (a + b)^2
`

`
a^2 - 2ab + b^2 = (a - b)^2
`

### Example
Factor:
`
x^2 + 6x + 9
`

Solution:
- Identify a = x, b = 3
- Apply the identity:

`
x^2 + 6x + 9 = (x + 3)^2
`
