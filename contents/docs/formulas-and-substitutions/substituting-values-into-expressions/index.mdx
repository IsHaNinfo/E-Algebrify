## Substituting Values into Expressions

### Concept

Substitution involves replacing the variables in an expression or formula with given numerical values.

### Steps

1. Identify the variables and their given values.
2. Replace each variable in the formula with the corresponding value.
3. Simplify to find the answer.

### Example 1

If `A = l × w`, find A when `l = 5`, `w = 3`.

* Substitute: `A = 5 × 3 = 15`

### Example 2

If `s = ut + 1/2at^2`, find `s` when `u = 2`, `t = 3`, `a = 4`.

* Substitute: `s = (2 × 3) + (1/2 × 4 × 9) = 6 + 18 = 24`
