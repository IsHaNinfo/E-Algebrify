## Solving Linear Equations in One Variable

To solve a linear equation, isolate the variable on one side of the equation.

### Example:
Solve:
`
3x - 5 = 7
`

Add 5 to both sides:
`
3x = 12
`

Divide both sides by 3:
`
x = 4
`
