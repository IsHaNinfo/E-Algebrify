## Basics of the Cartesian Plane


The Cartesian plane is a two-dimensional coordinate system where:
- The horizontal axis is called the x-axis
- The vertical axis is called the y-axis
- The point where they intersect is called the origin (0,0)
- Each point is represented by an ordered pair (x,y)

### Understanding Coordinates
- First number (x): distance from y-axis
- Second number (y): distance from x-axis
- Example: (3,2) means:
  * 3 units right on x-axis
  * 2 units up on y-axis
