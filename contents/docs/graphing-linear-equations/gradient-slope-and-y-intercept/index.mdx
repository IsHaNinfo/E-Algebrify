## Gradient and Y-intercept

### Understanding the Equation y = mx + c
- m is the gradient (slope)
- c is the y-intercept

### Gradient (m)
The gradient tells us:
- How steep the line is
- The rate of change
- The ratio of vertical change to horizontal change

Examples:
- m = 2: Line rises 2 units for every 1 unit right
- m = -1: Line falls 1 unit for every 1 unit right
- m = 0.5: Line rises 0.5 units for every 1 unit right

### Y-intercept (c)
The y-intercept tells us:
- Where the line crosses the y-axis
- The value of y when x = 0

Examples:
- c = 1: Line crosses y-axis at (0,1)
- c = -2: Line crosses y-axis at (0,-2)
- c = 0: Line passes through origin (0,0)
